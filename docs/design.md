# Design System

## Colors

- Background: #1a1208 (deep parchment-dark)
- Primary text: #E8D080 (antique gold)
- Secondary text: #C8B898 (warm parchment)
- Muted text: #7A6040 (aged ink)
- Panel background: rgba(20,14,8,0.93)
- Panel border: rgba(180,140,80,0.35)

## Location Type Colors

- City: bg #8B4513, border #FFD700
- Sacred: bg #6B21A8, border #E9D5FF
- Natural: bg #2D6A2D, border #90EE90
- Ruin: bg #5C4033, border #B8A398
- Landmark: bg #1E40AF, border #93C5FD

## Typography

- Main Font: Georgia, 'Times New Roman', serif (fantasy/parchment feel)

## Style Preferences

- Overall Style: Dark fantasy map aesthetic — aged parchment map, gold UI elements
- Border Radius: 6-8px for panels and buttons
- Animations: Minimal; marker size change on select
