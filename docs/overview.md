# Project Overview

## What is this?

An interactive fantasy world map for the Tramire continent — a rich fictional world. Users can explore locations, click markers to read lore descriptions, and reposition markers to correct placement.

## Key Features

- Zoomable, pannable map with the hand-drawn Tramire continent as background
- 40+ locations across all zones, each with name (Chinese + Latin), type, zone, and description
- Click a marker to open a detail panel on the right
- "Reposition" button in detail panel: click anywhere on map to move a marker
- Legend and zoom controls overlay
- Left-side sliding filter panel: filter by location type, faction (人类/树精/枝裔族/兽人/裂爪族), and zone

## Target Users

World-builders and writers working on the Tramire fantasy setting who need a visual reference map with lore annotations.
